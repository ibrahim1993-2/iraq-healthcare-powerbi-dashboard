Healthcare Infrastructure Trends in Iraq (1960–2024)

Overview
This project analyzes Iraq’s healthcare infrastructure trends over six decades, from 1960 to 2024.
It uses real-world historical data to visualize changes in:
- Hospital beds per 1,000 people
- Physicians per 1,000 people
- Year-over-Year (YoY) growth

The dashboard is fully interactive, allowing deep dives into healthcare trends with custom filters.

Data Sources
- World Health Organization (WHO)
- World Bank – Health infrastructure metrics

Tools Used
- Microsoft Power BI Desktop
- DAX (custom measures, YoY, Top N filters, error handling)
- Power Query for data cleaning
- Public datasets from WHO & World Bank

Dashboard Pages
Page 1 – Summary Overview
- Total hospital beds & physicians per 1,000 people
- Historical line trend visuals
- Slicers for Case Type and Year

Page 2 – Deeper Insights
- YoY growth visuals for both metrics
- Top N years view
- Average values across the dataset
- Dynamic Top N selector

Key Features
- Cleaned and transformed real-world historical data
- Trend analysis from 1960 to 2024
- DAX measures with “No data” handling
- Fully interactive visuals with filters and slicers

Repository Contents
- Healthcare Infrastructure Trends – Iraq (1960–2024).pbix (Power BI file)
- Dashboard Summary & Key Findings.pdf (project insights)
- Screenshots/ (Page 1 and Page 2 visuals)
- README.md (Markdown documentation)
- README.txt (this file)

How to Use
1. Open the `.pbix` file in Power BI Desktop
2. Use slicers to filter by Case Type or Year
3. Hover over visuals to view YoY changes
4. Visit Page 2 for deep dives and averages
